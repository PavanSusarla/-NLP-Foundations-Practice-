# 🧠 NLP Foundations Practice Repository

This repository contains my structured hands-on practice while building strong foundations in Natural Language Processing (NLP).

The goal is to master NLP from basics to advanced concepts before moving into Transformers and LLM Engineering.

---

## 📌 Learning Roadmap Covered

### 1️⃣ Text Pre-processing
- Tokenization
- Stopwords Removal
- Stemming (Porter, Snowball)
- Lemmatization
- Text Cleaning using Regex

### 2️⃣ Linguistic Processing
- POS Tagging
- Named Entity Recognition (NER)

### 3️⃣ Text Vectorization Techniques
- One Hot Encoding
- Bag of Words (BoW)
- TF-IDF
- N-grams

### 4️⃣ Word Embeddings
- Word2Vec
- CBOW & Skip-gram
- Average Word2Vec
- Semantic Similarity

### 5️⃣ Baseline Machine Learning
- Logistic Regression for Text Classification
- Feature Engineering with TF-IDF
